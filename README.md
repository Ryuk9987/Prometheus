# PROMETHEUS 🔥

> *Du bist ein Gott — aber kein allmächtiger. Du kannst nur flüstern.*

Ein God-Game / Zivilisations-Simulation, in dem der Spieler eine frühe Menschheit durch Ideen formt — nicht durch direkte Kontrolle.

## Konzept

Der Spieler kommuniziert via **Orakel** mit der Welt. Nur Gläubige empfangen die göttliche Stimme. Was sie daraus machen, liegt allein bei ihnen.

- 🧠 **Individuelle NPC-Simulation** — jeder Mensch hat eigenes Wissen, eigene Persönlichkeit
- 📜 **Wissenspropagation** — Wissen lebt in Menschen, stirbt mit ihnen (bis zur Schrift)
- 🙏 **Glaubenssystem** — ohne Glauben kein Orakelzugang
- 🌍 **Verformbare 3D-Welt** — NPCs hinterlassen physische Spuren
- ⚗️ **Emergente Technologie** — Bögen, Ackerbau, Schrift entstehen organisch
